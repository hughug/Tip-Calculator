# Tip Calculator
A tip calculator for ios platform.

## Main features
1. Compute the dinner cost plus tip and tax.
2. Compute each cost while splitting the check.

## Installation guide
To install, build the project using **[Xcode](https://developer.apple.com/xcode/)**.
